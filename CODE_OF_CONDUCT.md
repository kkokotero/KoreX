# Code of Conduct

## Our commitment

KoreX is meant to be a welcoming project for users, contributors, and maintainers.

We expect everyone involved to act with respect and to focus on improving the code and documentation.

## Expected behavior

- Be constructive.
- Be kind and direct.
- Focus feedback on the code, not the person.
- Keep discussions useful for the next contributor.
- Respect different experience levels.

## Unacceptable behavior

- Harassment
- Discrimination
- Personal attacks
- Trolling
- Intimidation
- Publishing private information without permission

## Enforcement

Project maintainers may remove, edit, or reject contributions that violate this code of conduct.

If you need to report a problem, use the private contact path described in the security policy or repository communication channels that support private disclosure.

## Scope

This code of conduct applies to all project spaces, including:

- issues
- pull requests
- discussions
- commit messages
- documentation
